​ 🛠️ IRON WEB | Creative Agency Portfolio
​IRON WEB is a high-end, high-conversion portfolio designed for the intersection of Business, Fashion, Education, and Marketing. Built with a minimalist, luxury aesthetic, it focuses on interactive user experience and direct lead generation.
​🚀 Live Features
​Glassmorphism UI: Modern, blurred backgrounds and sleek transitions.
​Industry-Specific Modals: 40+ specialized categories across Business, Fashion, Education, and Marketing.
​Dynamic Image Previews: Hover-triggered image updates within service modals.
​Smart Conversion: Integrated "Let's Build" inquiry form and direct WhatsApp/Social API links.
​Mobile Optimized: Fully responsive design for high-end mobile viewing.
​💻 Tech Stack
​HTML5: Semantic structure.
​Tailwind CSS: For luxury styling and rapid layout development.
​JavaScript (ES6): Modal logic, image switching, and form handling.
​Lucide Icons: Consistent, professional iconography.
